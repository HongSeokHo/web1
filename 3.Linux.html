Ahache2 설치

  • 1. 제일 먼저 서버 업그레이드부터 시행한다.

    • # sudo apt-get update -> 패키지 설치를 시작하기전에 저장소의 패키지 목록을 업데이트 해준다.

    # sudo apt-get upgrade -> 기존에 설치되어 있던 패키지를 업그레이드 해준다.

  • 2. Apache2 설치하기.

    • # sudo apt-get install apache2 -> Apache2 웹 서버를 설치한다.

    # sudo apt-get upgrade -> 패키지를 업그레이드한다.



    정상적으로 설치되었는지 확인하기 위해서, 웹브라우저를 실행하고 웹 주소를 localhost 또는 서버 IP 주소를 입력한다.

    웹을 실행시키고 그냥 "localhost"라고 입력해도 된다.



    위와 같은 웹페이지가 보이면, 정상적으로 아파치2가 설치된 것이다.



    >Apache2의 디폴트 Document Root는 /var/www/html 이다.

    이 경로에 HTML 또는 PHP파일 등을 생성하면 웹 브라우저를 통해 접근이 가능하다.

  • 3-1. 웹 홈페이지 IP로 접속하기



    • 두 대의 컴퓨터가 서로 웹 페이지를 주고 받기 위해서 먼저 필요한 것은 웹 서버가 설치된 컴퓨터의 주소이다.

    웹 서버 IP 확인법 : (터미널창에) hostname -i 를 입력하면, 자신의 ip가 나온다.



    위 그림처럼, local host가 아닌, 컴퓨터의 주소를 입력하여 웹 사이트에 접속이 가능하다.



    위 그림과 같이 나온다면, 서로 다른 컴퓨터가 정보를 주고 받는 것을 구현한 것이다 = 웹 서버를 구축한 것.

    >Index.html파일은 서버 컴퓨터(내 컴퓨터)의 >/var/www/html에 존재하고, 저 위치가 >웹 브라우저가 요청했을 때 웹 서버가 웹 페이지의 파일을 찾는 디렉토리이다.

    이 디렉토리에 HTML 또는 PHP 파일 등을 생성하면 웹 브라우저를 통해 접근이 가능하다.

    # /etc/apache2/apache2.conf 파일을 수정하여 디폴트 Documnet Root를 수정할 수 있다.

    MySQL 서버 설치

    MySQL은 관계형 데이터베이스 관리 시스템(relational database management system, RDBMS)이다.

    -> 다중 사용자가 다수의 데이터베이스에 접근할 수 있도록 서버를 구성하는것이다.

    # sudo -apt -get install mysql-server 명령어를 입력해서 mysql 서버를 설치한다.



    설치를 진행하다가 나중에 MySQL 서버의 관리자 계정인 root 사용자의 암호를 요구한다.

    사용할 암호를 입력하고, 기억해두자.
